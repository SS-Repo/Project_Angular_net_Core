Sistema Veicular Cadastrar

Exercicio:

Fazer uma tabela com os seguintes componentes:

  • Index - com uma tabela mostrando dados de um determinado veiculo com os seguintes dados:id,nome,placa e imagem do veiculo
  • Cadastro - cadastrar veiculo
  • Alterar - alterar veiculos
  • Remover - remover veiculo
  • Tecnologias exigidas:C# - NetCore,Angular,WebServices
